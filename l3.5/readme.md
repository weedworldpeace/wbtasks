## EventBooker 
    Сервис бронирования с дедлайнами

### 1. Запуск
    git clone github.com/weedworldpeace/wbtasks && cd l3.5 && docker compose up -d && make migrate-up

### 2. Описание
    Приложение состоит из основного сервиса, принимающего запросы, и воркера, подключенного к бд.
    Компоненты системы:

#### App Service (Порт: 8080)
    Принимает HTTP-запросы от пользователей
    Создаёт записи в PostgreSQL
    Отдает фронтенд
#### Worker Service
    Периодически ходит в БД и чистит истеченные бронирования
#### PostgreSQL (Порт: 5432)
    Хранит информацию о ивентах и бронированиях в двух связанных таблицах

### 3. API
    ФРОНТ /user - страница для пользователя
    ФРОНТ /admin - страница для админа
    POST   /api/v1/events      - создать ивент
    POST   /api/v1/events/{id}/book  - забронировать
    POST   /api/v1/events/{id}/confirm  - подтвердить бронь
    GET    /api/v1/events/{id} - получить информацию о ивенте
    GET    /api/v1/events/list - листинг всех ивентов

### 4. Интерфейс пользователя
    Веб-интерфейс позволяет:
    Просматривать ивенты
    Бронировать ивенты
    Подтверждать брони

### 5. Интерфейс админа
    Создание ивентов
    Просмотр бронирований
